## 🚀 Getting Started with These Prompts

### How to Use These Prompts

1. **Choose Your Repository**: Select the prompt section matching your current repository or task.
2. **Customize the Context**: Replace placeholders with your specific requirements, constraints, and goals.
3. **Iterate Evolutionarily**: Each AI response is a starting point — evolve it through multiple prompt iterations.
4. **Connect Across Repositories**: Use cross-repository prompts to ensure your work integrates with the broader ecosystem.
5. **Apply Many-Worlds Thinking**: Explore multiple prompt variations in parallel, then select the most fit result.

### Prompt Improvement Tips

- Add specific technology versions or constraints to get more precise outputs
- Include examples of desired output format to guide the AI response style
- Reference specific repositories by name to maintain ecosystem coherence
- Ask the AI to consider quantum/evolutionary principles when relevant
- Request retroactive analysis: "Given where we are, what should we have done differently?"

### Contributing New Prompts

To add prompts to this collection:
1. Follow the existing format with a descriptive heading and code block
2. Ensure the prompt aligns with the Evolutionary Evolutions Agency mission
3. Connect the prompt to relevant world communities and universe expansion goals
4. Include quantum, evolutionary, or eternity perspective where applicable
5. Submit via pull request with a description of the prompt's purpose and applications
