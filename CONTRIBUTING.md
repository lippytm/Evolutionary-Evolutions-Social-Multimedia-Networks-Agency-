# Contributing to Evolutionary Evolutions Agency

Thank you for your interest in contributing! 🎉

Every contribution — whether it's code, documentation, design, or ideas — helps grow this agency and its community.

---

## 🤝 How to Contribute

### 1. Reporting Issues / Requesting Features
- Open an [Issue](../../issues/new) and describe the problem or idea clearly
- Include steps to reproduce (for bugs) or use-case context (for features)

### 2. Improving Documentation
- Edit any `.md` file and open a Pull Request
- Fix typos, improve clarity, or add examples

### 3. Adding Code
1. Fork this repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes and commit: `git commit -m "Add: your description"`
4. Push to your fork: `git push origin feature/your-feature-name`
5. Open a Pull Request to `main`

### 4. Sharing Your Story
- Have you used this agency's resources to build something? We'd love to feature it!
- Open a [Discussion](../../discussions) and share your project

## 📋 Contribution Ideas

Looking for where to start? Here are some great first contributions:

- [ ] Add your favourite developer tool to `docs/REVENUE_GUIDE.md` (affiliate programme section)
- [ ] Translate any document into another language
- [ ] Add a new course idea to the content backlog in `docs/CONTENT_CALENDAR.md`
- [ ] Improve the README with better formatting or badges
- [ ] Create a template for a new type of GitHub repository in this agency

## 💖 Supporting Without Code

You don't have to write code to contribute:

- ⭐ **Star** this repository
- 💬 **Share** a post about this project on social media
- 💖 **[Sponsor](https://github.com/sponsors/lippytm)** the project
- 📣 **Refer** a business or developer who would benefit from this agency
