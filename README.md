## 🔄 Cross-Platform Integration

See [`cross-platform/`](./cross-platform/) for:
- `repo-index.json` — machine-readable index of all repositories and their roles
- `integration-map.json` — how each repository connects and communicates
- `api-bridge-config.json` — configuration for external platform APIs (ManyChat, BotBuilders, OpenAI, etc.)

---

## 🚀 Getting Started

1. **Clone this repo** as your local agency hub
2. **Review** `cross-platform/repo-index.json` to understand the full ecosystem
3. **Pick a language** from `ai-toolkit/languages/` and start building your AI agent
4. **Deploy a smart contract** using a template from `ai-toolkit/blockchain/`
5. **Connect your platforms** using `cross-platform/api-bridge-config.json`

## 📡 Automated Cross-Repo Sync

The `.github/workflows/cross-repo-sync.yml` workflow automatically:
- Pings the lippytm.ai hub on every push
- Reports repository status to the central integration map
- Can be triggered manually to broadcast updates across the ecosystem
