# 📅 Content Calendar — Evolutionary Evolutions Agency

## How to Use This Calendar

1. Copy the **Weekly Template** section each week
2. Fill in the topics from your content backlog
3. Schedule posts using a tool like [Buffer](https://buffer.com) or [Later](https://later.com)
4. Mark items complete and note performance metrics at week end

## 📋 Weekly Content Template

```
Week of: [DATE]
Theme: [MONTHLY THEME]
Goal: [e.g., grow YouTube by 50 subs, sell 5 courses]

MONDAY
  - [ ] GitHub: Merge/review PRs, respond to issues
  - [ ] Twitter/X: 1 tweet (tech tip or snippet)
  - [ ] LinkedIn: 1 post (business update or insight)

TUESDAY
  - [ ] Blog (dev.to/Hashnode): Publish or draft tutorial
  - [ ] Twitter/X: Thread (5–7 tweets on a single topic)
  - [ ] YouTube: Upload or schedule video

WEDNESDAY
  - [ ] GitHub: Open a new "good first issue" for contributors
  - [ ] Newsletter: Write draft
  - [ ] TikTok/Reels: Post 1 short-form video (30–60s demo)

THURSDAY
  - [ ] Twitter/X: Engage with 5 community members (replies, RTs)
  - [ ] LinkedIn: Share a project update or milestone
  - [ ] GitHub Discussions: Start or respond to a discussion

FRIDAY
  - [ ] Send newsletter
  - [ ] Twitter/X: Weekly recap tweet
  - [ ] YouTube Community post: Poll or sneak peek

WEEKEND
  - [ ] Schedule next week's social content
  - [ ] Review metrics (stars, views, open rate)
  - [ ] Brainstorm 3 new content ideas for backlog
```
