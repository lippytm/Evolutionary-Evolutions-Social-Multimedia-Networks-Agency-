## 1. 🤝 GitHub Sponsors

**Set up a GitHub Sponsors profile to accept recurring and one-time donations directly.**

### Setup Steps
1. Go to [github.com/sponsors](https://github.com/sponsors) and apply
2. Complete the [`.github/FUNDING.yml`](../.github/FUNDING.yml) file in this repo (already created)
3. Add a "Sponsor" button and badge to every repository README
4. Create tiered sponsorship packages:

| Tier | Monthly Price | Perks |
|---|---|---|
| ☕ Coffee | $5 | Name in CONTRIBUTORS.md |
| 🥉 Supporter | $15 | Early access to tutorials |
| 🥈 Collaborator | $50 | Monthly Q&A session, private Discord |
| 🥇 Partner | $100 | Logo in README, shoutouts, co-marketing |
| 💎 Enterprise | $500+ | Custom consulting hours, priority support |

### Tips
- Pin your Sponsors page link across all repos
- Thank sponsors publicly (with permission) in releases and social posts
- Create exclusive content for each tier to increase upgrades

## 5. 📺 Advertising Revenue

Monetise your audience through ad platforms.

### YouTube AdSense
- Enable monetisation at 1,000 subscribers + 4,000 watch hours
- Tech tutorials typically earn **$3–$10 CPM**
- 1 video × 10,000 views × $5 CPM = **$50 per video**
- Publish weekly → 52 videos/year → potential **$2,600–$26,000/year** at scale

### Blog / Dev.to
- [Carbon Ads](https://www.carbonads.net) — developer-focused, $1–$5 CPM
- [EthicalAds](https://ethicalads.io) — privacy-friendly, ~$2 CPM for tech content
- Self-hosted ads for direct sponsorships (higher rate)
